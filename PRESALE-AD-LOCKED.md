## STEP 2 — PASTE THIS AS THE FIRST MESSAGE

Using the attached True Vision Project logo and the attached product images, create a 20-second vertical (9:16) pre-sale ad for the TVP tracksuit drop. Use ONLY the attached images as the visuals — do not invent new garments. Build it in three phases with punchy, deliberate pacing:

**PHASE 1 — COMING (0–7s · the hook):** Open on black. At 0.5s hard-cut to the hidden-pocket close-up, snap-zoom in, and slam on bold condensed text: "THERE'S A POCKET IN THIS TRACKSUIT." Beat. Then type-on beneath it: "YOU'LL NEVER FIND IT." Add subtle scanning-UI overlays — thin crosshairs, "SEARCHING…" — over the fabric so it feels like a challenge. Fast jump-cuts, kinetic. Accent color pink.

**PHASE 2 — HERE (7–14s · reveal + the three):** Slow right down for contrast. Spotlight sweep across the hidden waistband image, elegant text fades in: "HIDDEN IN THE WAISTBAND. ONLY THE OWNER KNOWS." Transition to the three-colorway lineup; light each suit in sequence with its own accent — pink, ice blue, purple — then reveal: "THREE VERSIONS. ONE DROP." Premium, cinematic, smooth drift.

**PHASE 3 — GONE (14–20s · pre-sale close):** Cut to the PRE-ORDER hero (three models), slow push-in, overlay bold: "PRE-ORDER IS OPEN." then "LIMITED · NUMBERED · SECURE YOURS BEFORE IT'S GONE." Final 3s: black, the TV logo glows on in white, then in mono beneath: "THE WEBSITE IS LOCKED." → "TRUEVISIONPROJECT.COM" → "[ MEMBERS ENTER FIRST ]".

STYLE THROUGHOUT: near-black cinematic background, film grain, condensed bold display type for big lines + monospace for small technical lines, cream/white text with accent colors pulled ONLY from the garments (pink, ice blue, purple). Aggressive and fast in Phase 1, slow and premium in Phase 2–3. Built to work SILENT with captions — hard cuts, type-on and glitch reveals, no soft dissolves in Phase 1. End on a clean logo lockup.

## STEP 4 — FINISH + POST

2. Open in CapCut → add a TRENDING sound (browse trending, pick one that fits the beat cuts). Sound is ~40% of reach — don't skip it.
3. Make a SECOND version with a different first-2-seconds hook: "50 people will own something nobody can find." Same body. (A/B test.)
4. Post both, 7–9pm Irish time. 1–2 hashtags max.

## STEP 5 — READ THE SIGNALS (this decides the next move)
Watch 3 numbers, not likes:
- **3-second retention** — are they staying? >50% = strong hook.
- **Average watch time** — do they reach Phase 3?
- **Shares + saves** — the real viral currency.
Rule: if a version holds attention + gets shared → boost it €20–50. If it flops → the problem is the FIRST 2 SECONDS. Change only the hook and repost. 80% of the result is the opening.
